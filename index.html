Manual backup instructions

Use the shared SharePoint backup folder as the official source of truth.

  1. Open the HTML tracker from the shared SharePoint folder.
  2. Click Load Backup.
  3. Select the latest CURRENT_student_visa_tracker_backup.json file from the shared folder.
  4. Enter your name.
  5. Work normally during the day.
  6. At the end of the day, click Save Backup.
  7. Save one dated archive copy.
  8. Replace the shared CURRENT_student_visa_tracker_backup.json file with the newest backup.

This is a manual backup workflow, not a real-time multi-user database. Only one person should edit the tracker at a time to avoid overwriting another person's work.
